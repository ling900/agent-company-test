# Agent 公司 - 任务模板库

## 模板1：市场调研任务
```markdown
## 市场调研任务单

### 任务背景
[描述为什么要做这个调研]

### 调研目标
- [ ] 目标1
- [ ] 目标2

### 执行流程
1. 搜神：信息收集
   - 搜索渠道：[官网/社交/行业报告...]
   - 关键词：[列出关键词]
   
2. 凌风：分析与汇总
   - 整理关键发现
   - 制作对比表格
   
3. 汇报用户

### 预期产出
- 调研报告
- 关键数据整理
- 建议与结论
```

## 模板2：产品开发任务
```markdown
## 产品开发任务单

### 项目概述
[描述要做什么]

### 执行流程
1. 凌风：项目规划
   - 拆解功能模块
   - 制定开发计划
   - 分配任务
   
2. 构架：架构设计
   - 技术选型
   - 系统设计
   - 接口设计
   
3. 码灵：编码实现
   - 按模块开发
   - 单元测试
   - 代码审查
   
4. 凌风：集成测试
   - 部署上线
   - 验收测试

### 里程碑
- [ ] M1: 需求确认
- [ ] M2: 设计完成
- [ ] M3: 开发完成
- [ ] M4: 测试通过
- [ ] M5: 上线发布
```

## 模板3：代码审查任务
```markdown
## 代码审查任务单

### 代码来源
[仓库地址/文件路径]

### 审查要点
- [ ] 代码规范
- [ ] 安全漏洞
- [ ] 性能问题
- [ ] 可维护性

### 执行流程
1. 码灵：代码审查
   - 阅读代码
   - 运行测试
   - 提出改进建议
   
2. 凌风（如需要）：组织讨论
   - 汇总问题
   - 优先级排序
   
3. 汇报用户
```

## 模板4：多Agent协作讨论
```markdown
## 协作讨论记录

### 讨论主题
[要讨论的问题]

### 参与Agent
- [ ] 凌风
- [ ] 码灵
- [ ] 搜神
- [ ] 构架

### 讨论流程
1. 各自阐述观点
2. 方案对比分析
3. 协商确定最优解
4. 凌风拍板或上报用户

### 结论记录
[最终决策]
```
