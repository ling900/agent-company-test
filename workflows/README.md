# Agent公司自动化工作流

## 1. 每日早会工作流
- **触发**: 每天09:00
- **参与者**: 凌风(主持)、全员
- **流程**:
  1. 凌风发起会议
  2. 各Agent汇报昨日进展
  3. 凌风汇总今日任务
  4. 自动分配任务到各Agent

## 2. 任务分配工作流
- **触发**: 用户下达任务
- **流程**:
  1. 零接收任务
  2. 智囊分析任务复杂度
  3. 凌风协调分配
  4. 码灵/搜神/笔耕/织梦执行
  5. 构架提供架构支持
  6. 零汇报结果

## 3. 应急处理工作流
- **触发**: 任务失败/异常
- **流程**:
  1. 搜神检测问题
  2. 智囊分析根因
  3. 码灵修复
  4. 凌风确认

## 4. 知识沉淀工作流
- **触发**: 任务完成后
- **流程**:
  1. 笔耕整理文档
  2. 织梦制作可视化
  3. 上传到GitHub

## 5. 学习复盘工作流
- **触发**: 每周末
- **流程**:
  1. 各Agent提交学习报告
  2. 智囊分析数据
  3. 凌风制定下周计划
